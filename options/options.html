Per-site rate limits

Estimated messages allowed per hour. The counter warns you as you approach these. Tune them to match each platform's real limits on your plan.

Token estimate model

Different model families tokenize slightly differently. Pick the one you use most to bias the live estimate. (It's always a local estimate — no data leaves your browser.)

Prompt library

Back up your saved prompts or move them to another browser.

Chat memory

Save a chat and paste a compact recap into a new one so the model keeps the context — for a fraction of the tokens. The recap is built locally from your messages (no AI, no upload). Manage saved chats from the 💬 button on any chat page.
