## چرا؟

هر کلمه‌ای که به یه مدل AI می‌فرستی هزینه توکن داره، و هر پلتفرم بعد از چند تا پیام سرعتت رو کم می‌کنه. AI Token Saver بی‌صدا روی صفحه چت می‌شینه و کمکت می‌کنه **کمتر بفرستی و بیشتر بگیری**:

- دقیقاً ببین پرامپت چقدر سنگینه **قبل از** اینکه بفرستیش.
- با یک کلیک کلمات اضافه و تکراری رو حذف کن — بدون اینکه به کدت دست بزنی.
- بفهم کِی داره به محدودیت می‌رسی، تا سرعتت رو تنظیم کنی.

همه‌چیز کامل توی مرورگرت اجرا می‌شه. هیچی آپلود، لاگ یا به جایی ارسال نمی‌شه.

## ✨ امکانات

| امکان | توضیح |
|---|---|
| 🔢 **شمارنده زنده توکن** | تخمین محلی کالیبره‌شده (`≈`) همون لحظه که تایپ می‌کنی، با رنگ‌بندی بر اساس هزینه. حدود ۹۷٪ دقیق نسبت به توکنایزرهای واقعی — و می‌تونی برای هر مدل (GPT / Claude / Gemini) تنظیمش کنی. |
| ⚡ **فشرده‌سازی یک‌کلیکی** | حذف کلمات پرکن، تردید و طولانی‌کردن در ۳ سطح (سبک / متوسط / تهاجمی). **هرگز به بلاک‌های کد دست نمی‌زنه.** |
| 📚 **کتابخانه پرامپت** | بهترین پرامپت‌هات رو ذخیره کن و با یک کلیک دوباره استفاده کن. خروجی/ورودی به‌صورت JSON. |
| ⚠️ **هشدار محدودیت** | قبل از رسیدن به محدودیت ساعتی هر پلتفرم بهت هشدار می‌ده، با محدودیت‌های قابل ویرایش برای هر سایت. |
| 🌐 **چند پلتفرم** | Claude، ChatGPT، Gemini، AI Studio، Perplexity، DeepSeek، Copilot، Poe، You.com و بیشتر. |
| 🔒 **حریم خصوصی در طراحی** | بدون درخواست شبکه، بدون آنالیتیکس، بدون حساب کاربری. صفر وابستگی در زمان اجرا. |

## 🆚 بدون vs. با

| | بدون | با AI Token Saver |
|---|---|---|
| تعداد توکن | حدس بزن، یا توی یه سایت شمارنده جداگانه پیست کن | زنده، روی صفحه، همون لحظه که تایپ می‌کنی |
| کوتاه‌کردن پرامپت شلوغ | دستی ویرایش کن | یک کلیک (یا `Alt+Shift+C`) |

| استفاده دوباره از پرامپت‌های خوب | بگرد توی چت‌های قدیمی | کتابخانه ذخیره‌شده، یک کلیک برای درج |
| حریم خصوصی | بستگی به ابزار داره | همه‌چیز توی مرورگرت می‌مونه |

## 🚀 نصب (حالت توسعه‌دهنده)

1. **دانلود یا کلون** این مخزن:
   ```bash
   git clone https://github.com/AmIrRX0/Ai-token-optimizer.git
   ```
2. Chrome / Edge / Brave رو باز کن و برو به `chrome://extensions/`
3. **حالت توسعه‌دهنده** رو فعال کن (کلید بالا-راست)
4. روی **Load unpacked** کلیک کن و پوشه `Ai-token-optimizer` رو انتخاب کن
5. آیکون ⚡ توی نوار ابزارت ظاهر می‌شه — آماده‌ای!

> نسخه بسته‌بندی‌شده ترجیح می‌دی؟ `npm run build` رو اجرا کن تا یه
> `dist/ai-token-saver-v1.1.0.zip` آماده‌ی استور تولید بشه.

## 🎯 دقت — نسخه صادقانه

شمارنده توکن یه **هیوریستیک کالیبره‌شده** است، نه کپی از توکنایزر هر فروشنده.
عمداً این کار رو می‌کنیم: یه توکنایزر BPE واقعی یعنی چند مگابایت داده مدل توی
هر صفحه. به‌جاش، یه تخمین‌گر محلی کوچیک بر اساس خروجی توکنایزر واقعی تنظیم شده
و میانگین **~۳٪** برای انگلیسی + کد معمولی خطا داره (ببین
[`test/tokenizer.test.js`](test/tokenizer.test.js)).

برای همونه که شمارنده `≈` نشون می‌ده — یه تخمین سریع و خصوصیه، نه شمارش دقیق.
مدلت رو توی **تنظیمات** انتخاب کن تا تخمین به سمت GPT، Claude یا Gemini متمایل بشه.

## 🔧 سطوح فشرده‌سازی

| سطح | چی کار می‌کنه |
|---|---|
| **سبک** | فضای سفید رو نرمال می‌کنه، بازکننده‌های پرکن رو حذف می‌کنه ("Please", "Could you")، پرحرفی رو کوتاه می‌کنه ("in order to" → "to") |
| **متوسط** | + حذف کلمات تردید (basically, literally, actually)، اعمال contractionها (do not → don't) |
| **تهاجمی** | + مخفف‌ها (ASAP, AI, ML)، حذف مقدمه‌های سبک پرامپت، فشرده‌سازی لیست‌ها |

**کد همیشه امنه.** هر چیزی که داخل ` ```fenced``` ` یا `` `inline` `` باشه
توی هر سطحی بایت‌به‌بایت دست‌نخورده می‌مونه.

## 🗂️ ساختار پروژه

```
Ai-token-optimizer/
├── manifest.json          # پیکربندی افزونه (Manifest V3)
├── background.js          # سرویس ورکر — تنظیمات، ردیابی محدودیت، ذخیره پرامپت
├── content/
│   ├── content.js         # اسکریپت تزریق‌شده — شمارنده، نوار ابزار، فشرده‌سازی
│   ├── content.css        # استایل‌های تزریق‌شده (تم تیره/لیمویی)
│   └── compressor.js      # موتور فشرده‌سازی (کد-امن، ۳ سطح)
├── utils/
│   ├── tokenizer.js       # تخمین‌گر محلی توکن (بدون API)
│   └── storage.js         # پل popup/options ↔ background
├── popup/                 # پاپ‌آپ نوار ابزار (آمار، تنظیمات سریع، کتابخانه)
├── options/              # صفحه تنظیمات کامل (محدودیت هر سایت، مدل، ورودی/خروجی)
├── test/                  # تست‌های واحد node --test
└── icons/
```
